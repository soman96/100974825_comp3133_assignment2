Employee Management System

---

## Technologies Used

- Node.js
- Express
- Apollo Server (GraphQL)
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Cloudinary (for employee photos)

---

## Installation

1. Clone the repo
2. Install dependencies
3. Create a `.env` file in the root folder with the following:


MONGO_URI=
PORT=4000
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

4. Run server!
